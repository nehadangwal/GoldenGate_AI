Security Policy
Supported Versions
SentryNode is currently in Active Development (v0.1.0). We prioritize security for the latest main-branch releases.

Reporting a Vulnerability
Do not report security vulnerabilities through public GitHub issues.

If you discover a security vulnerability—specifically regarding Adaptive Routing bypasses or Sovereign Execution leaks—please report it privately by contacting Neha Dangwal via LinkedIn.

We aim to acknowledge all reports within 48 hours and provide a fix or mitigation strategy for validated issues.
